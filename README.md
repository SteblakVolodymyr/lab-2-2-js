# ЛР 2.1 + ЛР 2.2  
## Тема: Моделювання руху тіла, кинутого під кутом до горизонту

## Вміст архіву
- `docs/LR2_1_Methodology.docx` — технічне завдання, вибір методології Kanban, графічний алгоритм.
- `docs/LR2_2_Code_Testing.docx` — опис реалізації, структура проєкту, тестування.
- `src/index.html` — головна сторінка застосунку.
- `src/css/style.css` — стилі інтерфейсу.
- `src/js/app.js` — логіка обчислення, валідація, таблиця та графік.
- `assets/flowchart.png` — графічний алгоритм.

## Як запустити
1. Розпакувати архів.
2. Відкрити файл `src/index.html` у браузері.
3. Ввести початкові дані.
4. Натиснути кнопку `Розрахувати`.

## Основні можливості програми
- обчислення часу польоту;
- обчислення дальності польоту;
- обчислення максимальної висоти;
- побудова таблиці координат;
- побудова графіка траєкторії;
- перевірка помилок введення.
